这是相对稳妥的方式。

## 三、找人代绑手机号验证：强烈不推荐

市面上还有一些所谓的“代验证”或“代绑手机号”服务，表面上看可以解决当前的登录验证，但这种方式存在一个非常严重的隐患：

**帮助你接收验证码的手机号，通常并不真正属于你。**

第一次验证成功，并不代表以后还能继续使用。一旦账号再次触发手机号验证，原来的代绑号码可能已经被回收、停用，或者绑定了其他账号。

这时你就无法再次接收验证码，严重情况下，账号可能长期无法登录。

更需要警惕的是，部分代绑服务可能重复使用手机号，甚至保留验证记录。除了二次验证失败外，还可能带来账号找回、隐私泄露和账号被盗等风险。

因此，不要只看眼前能不能登录。对于准备长期使用的账号来说，手机号是否能够持续控制，比第一次验证成功更加重要。
